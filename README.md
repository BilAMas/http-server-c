# Toy HTTP Server (C)

This project is a toy HTTP/0.9 and HTTP/1.0 server written in C to explore the fundamentals of network programming and the HTTP protocol. The was to learn the inner workings of the protocol and translate the official RFC into actionable tasks and code.
